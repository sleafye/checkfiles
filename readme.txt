1. windows
checkfiles_win.exe -o 工程文件夹相对路径 -t 比对文件夹相对路径
例: checkfiles.exe -o WLSCMJGame -t XYCppGame-mobile

2. mac
./checkfiles_mac -o 工程文件夹相对路径 -t 比对文件夹相对路径
例: ./checkfiles_mac -o WLSCMJGame -t XYCppGame-mobile

mac要开权限 sudo chmod -R 777 文件路径
提示“macOS无法验证此App不包含恶意软件” 在设置->安全性与隐私->通用 开启应用
